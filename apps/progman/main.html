Help

Welcome to the Program Manager in Windows 11 style. This application allows you to create shortcuts to ZIP files containing executable applications. Usage instructions: 1. To install a new application: - Click on "File" > "Install" - Select a ZIP file - Assign a name to the shortcut - Optionally, select a custom icon - If no icon is provided, a generic Windows 11 style app icon will be assigned 2. To uninstall an application: - Click on "File" > "Uninstall" - Select the shortcut you want to remove 3. To run an application: - Click on the shortcut on the main screen - Or go to "File" > "Run..." and manually select a ZIP file 4. To move a shortcut: - Click and hold on the icon and drag it to the desired position 5. Winefile: - The "Winefile" icon is a permanent shortcut that allows you to explore files - You can run ZIP files directly from this explorer ZIP file format: - Each ZIP file must contain an executable file with the same name as the ZIP file, but with the .exe extension - For example, if the ZIP file is called "myapp.zip", it must contain a file called "myapp.exe" - If the ZIP file does not contain the corresponding executable, an error message will be displayed indicating that the file is not compatible Created shortcuts will be automatically saved and will appear each time you open the application.
