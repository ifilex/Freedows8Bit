Ayuda - Contenido

Esta aplicación emula el estilo del Program Manager de Windows 3.1, donde puedes instalar y ejecutar programas ZIP.

Formato del archivo ZIP:

  • Debe contener un archivo ejecutable (.exe).
  • El nombre del archivo ZIP y del archivo .exe deben coincidir.
