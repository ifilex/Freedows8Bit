WineBox – Cognitive & Emulation Environment (Legacy + ARM)

This repository contains both the historical Windows-like emulator (Freedows8Bit /
Wine-Proton APK) and the new WineBox ARM Cognitive System.

Legacy versions:
- Wine-Proton APK (preserved binary)
- Termux runtime (Android → Termux → proot → X11 → Box64 → Proton)

Next-gen version:
- ARM Linux OS
- DEB module system
- Cognitive modules for neurodevelopment
- Secure patient session sandbox

All legacy content is preserved for historical and technical reference.
