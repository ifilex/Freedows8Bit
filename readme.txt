Glass Program Manager v 3.0

Glass​ es una aplicación de simulación de escritorio de código abierto para dispositivos Android 
actualmente en desarrollo que se centra específicamente en la informática personal y multimedia. 
Inspirado en el Program Manager de Windows 3.11, Glass es un sistema rápido, eficiente, fácil de 
aprender y usar, sin descuidar su potencia para los usuarios de todos los niveles. Actualmente se 
encuentra en fase beta: soportando una compatibilidad centrada en Windows 8.1 permitiendo por
medio de las librerias Wine, y el proyecto ReactOS ejecutar todo el contenido de programas de
Windows sin utilizar ninguna parte de este.


Por hacer:

- Crear un escritorio de desarrollo con utilidades para usuarios y programadores
- Revisar fallas y optimizar el codigo.


En desarrollo:

* Creador de APK para distribuir aplicaciones windows en 32 y 64 bit soportando aceleracion  grafica que se
podra configurar cuando se genere el contenedor de datos que permitira ejecutar las aplicaciones.

Novedades de esta version

- Re diseño del entorno e inicio del desarrollo del codigo android para creacion de apk que puedan distribuirse
de forma libre para facilitar el port de aplicaciones y juegos a la plataforma android.
- Se retiro todas las apps que eran HTML5 por aplicaciones unicamente windows.
- Inicio del desarrollo del kernel 64 bit para soporte de windows 64 bit.


Resumen:

Simulador de escritorio con la capacidad de ejecutar aplicaciones, juegos y contenido multimedia 
dentro de un entorno web.
